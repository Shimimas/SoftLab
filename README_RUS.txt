Репозиторий содержит каталог src. В котором содержатся папки, соотвественно заданиям в формате
n(номер задания).
Также в каталоге содержится Makefile с целями:
n6 - запускает программу, в которую можно вводить данные
test6 - запускает тесты
n6_clean - очищает директорию от объектов сборки
(6 можно заменить на любой номер задания от 6-9)
Для удобства каждое решение разбито на 5 файлов:
- *.h - заголовочный файл, в котором также содержится анализ эффективности алгоритма
- test.cpp - тесты
- foo.cpp - функции для решения задачи
- n.cpp - для того, чтобы запустить программу
- answer.cpp - для того, чтобы проверяющий легко мог отправить решение в тестирующую систему сайта
